## E-Commerce Analysis
### 1. Objective 
#### Analyze sales and its trends through discounts, delivery performance, returns, and customer behavior and find solutions to increase and improve it.
### 2. Data Model 
####Fact tables: fact_order_items, fact_returns. Dimension tables: dim_customer, dim_product, dim_date, dim_courier.
### 3. Hypothesis Testing 
####• Discounts significantly increase sales. 
#### • Longer delivery time increases return probability. 
#### • Courier performance differs significantly. 
#### • Return behavior varies by product category. 
### 4. Tools 
#### SQL Server, Power BI, Python (Pandas, Matplotlib, Seaborn, sqlalchemy, Faker, Statsmodels).
